# 📊 Customer Behaviour Data Analysis

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior to extract meaningful business insights.
The goal is to understand purchasing patterns, customer segmentation, and revenue trends using **SQL, Python, and Power BI**.

---

## 🎯 Objectives

* Analyze customer spending behavior
* Identify top-performing products
* Evaluate the impact of discounts and subscriptions
* Segment customers based on purchase history
* Compare revenue across demographics (gender & age groups)

---

## 🛠️ Tools & Technologies Used

* **SQL (MySQL)** – Data querying & analysis
* **Python (Pandas)** – Data cleaning & preprocessing
* **Microsoft Excel** – Initial data handling
* **Power BI** – Dashboard creation & visualization

---

## 🐍 Python Analysis

Performed data preprocessing and exploratory data analysis using Pandas:

* Handling missing values and data cleaning
* Creating new features (age groups, customer segments)
* Performing exploratory data analysis (EDA)
* Preparing dataset for SQL and Power BI

---

## 📊 Key SQL Analysis

* **Revenue by Gender** – Compared total revenue from male vs female customers
* **High Spending Customers with Discounts** – Identified customers spending above average despite discounts
* **Top Rated Products** – Extracted top 5 products based on ratings
* **Shipping Type Analysis** – Compared Standard vs Express shipping impact
* **Subscription Impact** – Compared spending behavior of subscribed users
* **Discount Effectiveness** – Measured discount usage across products
* **Customer Segmentation** – Categorized customers into New, Returning, Loyal
* **Top Products by Category** – Identified best-selling items per category
* **Repeat Buyer Analysis** – Analyzed subscription trends among repeat buyers
* **Revenue by Age Group** – Compared revenue contribution across age segments

---
